Markdown 简明教程

2021-05-26

Markdown 是一种轻量级标记语言。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。在很多网站得到了广泛的应用，比如说 GitHub 知乎 简书等，人们使用它发表文章，你现在看到的这一篇文章就是使用 Markdown 创作的。

标题

语法：使用 # 表示标题，1个#代表一级标题，2个#代表二级标题，#后面有一个空格，以此类推，最多六级标题。

实例：# 这是一个一级标题

预览：

字体

粗体

语法：使用 **文本**表示粗体

实例：**这是一行粗体**

预览：

这是一行粗体

斜体

语法：使用*文本*表示斜体

实例：*这是一行斜体*

预览：

这是一行斜体

粗体+斜体

语法：使用***文本***表示粗体+斜体

实例：***这一行是粗体+斜体***

预览：

这一行是粗体+斜体

删除线

语法：使用~~文本~~表示删除线

实例：~~这一行是删除线~~

预览：

这一行是删除线

下划线

语法：markdown 本身不支持下划线，但可以使用 HTML 中的<u>标签实现

实例：<u>这一行是下划线</u>

预览：

这一行是下划线

分割线

语法：可以使用3个以上的 * - _ 表示分割线。行内可以插入空格，但不能有其他东西。

实例：***

预览：

列表

无序列表

语法：使用* + -表示无序列表，* + -后面有一个空格

实例：

+ 这是一行无序列表1
+ 这是一行无序列表2
+ 这是一行无序列表3

预览：

  • 这是一行无序列表1
  • 这是一行无序列表2
  • 这是一行无序列表3

有序列表

语法：使用 阿拉伯数字 + . 表示有序列表， .后面有一个空格

实例：

1. 这是一行有序列表1
2. 这是一行有序列表2
3. 这是一行有序列表3

预览：

  1. 这是一行有序列表1
  2. 这是一行有序列表2
  3. 这是一行有序列表3

列表嵌套

语法：只需要在子列表前加上4个空格即可

实例：

+ 第一项
    1. 子列表1
    2. 子列表2
    3. 子列表3
+ 第二项
    1. 子列表1
    2. 子列表2
    3. 子列表3
+ 第三项
    1. 子列表1
    2. 子列表2
    3. 子列表3

预览：

  • 第一项
    1. 子列表1
    2. 子列表2
    3. 子列表3
  • 第二项
    1. 子列表1
    2. 子列表2
    3. 子列表3
  • 第三项
    1. 子列表1
    2. 子列表2
    3. 子列表3

引用

引用

语法：在引用的段落开头加上>，>后面有一个空格

实例：> 这一行是引用的文本

预览：

这一行是引用的文本

引用嵌套

语法：1个 > 是第1层（即最外层），2个 > 是第二层，以此类推

实例：

> 最外层
>> 第二层 
>>> 第三层

预览：

最外层

第二层

第三层

代码

代码

语法：使用 Tab 键 上面的 ` 键 表示代码

实例：

`Hello World!`

预览：

Hello World!

代码块

语法：使用 3 个 Tab 键 上面的 ` 键 表示代码块，可以指定编程语言

实例：

预览：

$(document).ready(function () {
    alert('Hello World!');
});

链接

链接

语法：[链接描述] +(链接地址)表示链接，链接描述可以为空，链接地址可以是外链，也可以是本地文件地址

实例：

[Jianrry 的博客](https://blog.jianrry.com)

预览：

Jianrry 的博客

高级链接

语法：[链接描述][链接变量名]表示链接，通过变量设置一个链接，链接变量一般在文章末尾赋值

实例：

欢迎访问我的博客：[Jianrry 的博客][jianrry]

[jianrry]: https://blog.jianrry.com

预览：

欢迎访问我的博客：Jianrry 的博客

图片

语法：! + [图片描述] + (图片地址)表示图片，图片地址可以为空，图片地址可以是外链，也可以是本地图片的地址

实例：

![Jianrry 的头像](https://drive.jianrry.com/image/avatar.jpg?raw)

预览：

表格

语法：使用 -分割表头和普通行，使用|分割单元格

:---- 表示左对齐
----: 表示右对齐
:----: 表示居中

实例：

| 姓名| 性别 | 年龄 | 班级 |
| ---- | ---- | ---- | ---- |
| 张三 | 男 | 12 | 初一(1)班 |
| 李四 | 男 | 13 | 初二(1)班 |
| 王五 | 男 | 14 | 初三(3)班 |

预览：

姓名 性别 年龄 班级
张三 男 12 初一(1)班
李四 男 13 初二(1)班
王五 男 14 初三(3)班
