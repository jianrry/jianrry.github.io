Windows 常用键盘快捷键

2021-05-29

本文介绍了 Windows 常用的键盘快捷键。

    • Ctrl 键
    • Windows 键
    • 其他键
    • 参考资料

Ctrl 键

快捷键 描述
Ctrl + A 全选
Ctrl + C 复制
Ctrl + V 黏贴
Ctrl + Z 撤销
Ctrl + S 保存
Ctrl + F 搜索
Ctrl + H 替换
Ctrl + N 打开新窗口
Ctrl + R 刷新活动窗口
Ctrl + W 关闭活动窗口
Ctrl + D 移动到回收站

Windows 键

快捷键 描述
Windows 打开或关闭“开始”菜单
Windows + D 显示或隐藏桌面
Windows + E 打开文件资源管理器
Windows + G 打开游戏栏（当游戏处于打开状态时）
Windows + Alt + R 开始或停止录制
Windows + I 打开设置
Windows + L 锁定你的电脑或切换帐户
Windows + R 打开“运行”对话框
Windows + V 打开剪贴板

其他键

快捷键 描述
CapsLk 切换英文大小写
Num Lock 打开或关闭小键盘锁
Shift + CapsLk 切换输入法
PrtSc 捕获整个屏幕的屏幕截图并将其复制到剪贴板
