注册域名

Step1：登陆你的 Freenom 账号

打开官网，登录你的 Freenom 账号。

Freenom 不开放注册，请使用 Google 或 Facebook 账号登陆。

Step2：搜索你想注册的域名

打开官网，搜索你想注册的域名。

输入搜索的关键字，然后点击“Check Availability”，检查可用性。

Step3：将域名加入购物车

点击“Get it now!”按钮，将域名加入购物车。

Step4：确认订单

点击“Checkout”按钮，进入订单页面。

默认是 3 Months @ FREE （免费3个月），你可以更改为 12 Months @ FREE（免费12个月）。

Step5：完成订单

勾选上“ I have read and agree to the Terms & Conditions "，然后点击" Complete Order " ，完成订单。

Step6：注册成功

如果出现了下面的提示，则说明已经注册成功。

域名解析

Step1：访问域名管理页面

点击此处，访问域名管理页面。

点击 "Manage Domain" 按钮，管理你的域名。

Step2：修改 Name Server

点击 "Management Tools" 按钮，继续点击 "Nameservers" 按钮，进入 Name Server 管理页面。

选中 "Use custom nameservers(enter below)"，自定义 Name Server。

输入 Name Server 的地址，然后点击 "Change Nameservers"。

域名续期

在你的域名到期前14天，才可以免费续期。一次最多可以续期12个月，没有次数限制。

点击此处，访问域名续期页面。

GitHub：https://github.com/luolongfei/freenom

建议你使用上面的 GitHub 项目，实现 Freenom 域名自动续期。

注意事项

  1. 国内的 IP 已经无法注册 Freenom 的域名，建议使用美国 IP 进行注册。
  2. 你的 Freenom 账户上的地址需要和你的 IP 地址的所在地 保持一致。
