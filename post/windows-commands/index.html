打开命令提示符的方式

  1. 同时按下“Windows + R”键，打开运行对话框，输入“cmd”，然后回车（推荐）。
  2. 依次点击“开始菜单-Windows系统-命令提示符”。
  3. 在任意文件夹中的空白区域，同时按下 "Shift" 键和鼠标右键，然后点击“在此次打开PowerShell窗口”。
  4. 在文件资源管理器的地址栏前加上 "cmd "，"cmd"后面有一个空格，然后回车。

常用命令

dir

说明：directory（目录）的缩写。显示目录的文件和子目录的列表

语法：

dir [<drive>:][<path>][<filename>]

实例：

dir

预览：

PS C:\Users\admin\Desktop> dir


    目录: C:\Users\admin\Desktop


Mode                 LastWriteTime         Length Name
----                 -------------         ------ ----
-a----         2021/5/22     16:09           1609 electerm.lnk
-a----         2021/5/28     17:05           1050 Notepad++.lnk
-a----         2021/5/27     17:13           1385 v2rayN.lnk

md

说明：make directory（创建目录）的缩写。创建目录或子目录。

语法：

md [<drive>:]<path>

实例：

md Directory1

预览：

PS C:\Users\admin\Desktop> dir


    目录: C:\Users\admin\Desktop


Mode                 LastWriteTime         Length Name
----                 -------------         ------ ----
d-----         2021/5/30     15:01                Directory1
-a----         2021/5/22     16:09           1609 electerm.lnk
-a----         2021/5/28     17:05           1050 Notepad++.lnk
-a----         2021/5/27     17:13           1385 v2rayN.lnk

rd

说明：remove directory（删除目录）的缩写。删除一个空目录。

语法：

rd [/s] [/q] [<drive>:]<path>

实例：

rd Directory1

预览：

PS C:\Users\admin\Desktop> dir


    目录: C:\Users\admin\Desktop


Mode                 LastWriteTime         Length Name
----                 -------------         ------ ----
-a----         2021/5/22     16:09           1609 electerm.lnk
-a----         2021/5/28     17:05           1050 Notepad++.lnk
-a----         2021/5/27     17:13           1385 v2rayN.lnk

备注：

/s ：删除 (指定目录及其所有子目录（包括) 的所有文件）的目录树。

/q：指定安静模式。 删除目录树时不提示进行确认。

cd

说明：change directory（改变目录）的缩写。显示当前目录的名称或更改当前目录。

语法：

cd [<drive>:][<path>]

实例：

cd C:\Users\admin\Desktop

预览：

备注：

如下所示，带有空格的路径，需要加上双引号。

cd C:\"Program Files"

cd 命令还有另一种用法，可以用于创建一个空文件。

 cd . > filename.txt

del

说明：delete（删除）的缩写。删除一个或多个文件。

语法：

del [/f] [/s] [/q] <names>

实例：

del filename.txt

预览：

备注：

/f ：强制删除只读文件。
/s：删除当前目录和所有子目录中的指定文件。
/q：安静模式。 不会提示您确认删除。

cls

说明：clear screen（清除屏幕）的缩写。清除 "命令提示符" 窗口。

语法：

cls

实例：

cls

预览：

ping

说明：通过向另一台 TCP/IP 计算机发送 Internet 控制消息协议 (ICMP) 回响请求消息来验证 IP 级连接。

语法：

ping <targetname>

实例：

ping blog.jianrry.com

预览：

ipconfig

说明：显示所有当前 TCP/IP 网络配置值并刷新动态主机配置协议 (DHCP) 和域名系统 (DNS) 设置。 在没有参数的情况下使用， ipconfig 显示 Internet 协议版本 4 (IPv4) 以及所有适配器的 IPv6 地址、子网掩码和默认网关。

语法：

ipconfig

实例：

ipconfig

预览：

PS C:\Users\admin\Desktop> ipconfig

备注：

ipconfig 还有一个常见的用法，可以用于刷新 DNS 缓存。

ipconfig /flushdns
