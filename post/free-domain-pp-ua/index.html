免费注册pp.ua域名

2021-05-10

.ua 是乌克兰的国家域名后缀，而 pp.ua 中的 pp 是 private person（私人 个人） 的缩写。pp.ua 是乌克兰官方为个人提供的免费二级域名后缀，任何人都可以申请。

注册域名

Step1：登陆你的 NIC.UA 账号

打开官网，登录你的 NIC.UA 账号（如果没有，就注册一个 NIC.UA 账号）。

Step2：搜索你想注册的域名

打开官网，搜索你想注册的域名。

Step3：将域名加入购物车

点击“Add to cart”按钮，将域名加入购物车。

Step4：确认订单

点击“Checkout”按钮，进入订单页面。

点击“Continue”按钮，确认订单。

Step5：填写个人信息

建议如实填写你的个人信息，其中邮箱和手机号必须是真实的，其他的信息尽量填写真实的。

Step6：注册成功

点击“Continue checkout”按钮，如果出现了下面的提示，则说明已经注册成功。

Step7：检查邮件

你会收到2封邮件，复制里面的链接在浏览器中打开即可。

如果没有收到邮件，请检查一下垃圾箱。

域名激活

Step1：获取验证码

方法1（推荐）：

点击此处， 向 pp.ua 的官方 Telegram 机器人(@ppuabot) 发送消息。

方法2：

等待官方向你的手机号发送短信验证码，可能存在延迟，需要等待几天时间。

Step2：激活域名

点击此处，依次输入你的域名、手机号、验证码。完成人机验证后，点击“Continue”按钮进行提交。

如果出现了下面的提示，则说明激活成功。

域名解析

Step1：修改 NS 服务器

访问域名页面，点击右侧的齿轮图标。

滑动到下面，将 NS-servers 的下拉选项改为 NIC.UA name servers ，然后点击“Change NS”按钮。

Step2：添加 DNS 解析

访问 NS服务器页面，点击右侧的齿轮图标。

然后点击 "Change"，添加或修改 DNS 记录。

注意事项

  1. 在30天内，最多只能注册3个.PP.UA后缀的域名。
  2. 域名的有效期为一年，一年免费续订一次，到期前60天才能续订。
  3. 如果域名到期后没有续费，在到期后的28天内可以花费996 UAH，赎回你的域名。
  4. 不提供 WHOIS 保护，你的个人信息始终对外公开。
